Propriedades CSS – Resumo Visual

Propriedade Para que serve
color Cor do texto
background-color Cor de fundo
font-size Tamanho da letra
font-family Fonte do texto
text-align Alinhamento do texto
padding Espaço interno do elemento
margin Espaço externo ao redor do elemento
border Adiciona borda ao redor
width / height Largura e altura do elemento
display Tipo de exibição (ex: block, flex, inline)
position Posicionamento (relative, absolute, fixed...)
box-shadow Sombra em volta da caixa
text-shadow Sombra no texto
transition Animação suave de mudanças
:hover Estilo ao passar o mouse sobre o elemento
